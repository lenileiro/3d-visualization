# React Express Boilerplate

# Install
``` 
    npm install
```

# Development

#### To build the project into static files

``` 
    npm run build:client
```

#### Run client on develpment mode

```
    npm run start:client
```

### Serve side render the react application

```
    npm run start:server
```
